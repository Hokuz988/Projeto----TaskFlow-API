📋 Informações da API

Versão: 1.0.0

Formato: JSON

Licença: MIT

Especificação: OpenAPI 3.0.3

🔐 Autenticação

Para acessar os endpoints protegidos, inclua o header:

Authorization: Bearer {seu-token}

Tipo: Bearer Token (Laravel Sanctum)

Como obter token: Faça login através do endpoint de autenticação.

👥 Usuários: Autenticação segura com Laravel Sanctum
